MIDI Values Chart

A MIDI controller can have up to 128 (8 bit) distinct pitches. This table shows them and the corresponding English musical note names in a concise chart for reference. This cheat sheet for musicians provides a handy means to quickly look up midi note numbers and see which notes and octave numbers correspond to them.
