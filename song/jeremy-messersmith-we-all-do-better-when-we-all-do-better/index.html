Jeremy Messersmith - We All Do Better When We All Do Better

For Educational Use. Optimized for GCEA Ukulele. Key of C: C, F, G7.

As performed by jeremy messersmith.

The original does not use a 7th chord and is in the Key of F. This version has been transposed for beginner students, and uses a G7 chord.

The ukulele chord font matches Aquila's KIDS strings (available in bulk sets of 20 directly from Aquila) and the font is a part of Chordette for Education, available at ukefarm.com
